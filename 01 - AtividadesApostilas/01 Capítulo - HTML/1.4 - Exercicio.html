Exercício: Descreva a diferença entre uma página de hipertexto estática e uma dinâmica, pontuando as vantagens e desvantagens de cada um dos dois tipos.

Uma página estática é aquela que é entregue ao navegador do usuário exatamente como está armazenada no servidor. Geralmente é escrita em HTML puro, com CSS e, possivelmente, JavaScript para formatação e interatividade básica. Como vantagem podemos citar a simplicidade e facilidade de desenvolvimento além da velocidade, uma vez que ela não precisa de recursos pesados de processamento. Como desvantagem podemos citar manutenção, riqueza de recursos e conteúdos para serem explorado.

Já a páginas dinâmicas são geradas em tempo real, geralmente com base em interações do usuário, dados de um banco de dados, ou entrada de um formulário. Elas utilizam linguagens de programação do lado do servidor para criar conteúdo que muda dinamicamente. Como vantagem podemos citar personalização, facilidade de atualização e funcionalidades avançadas. Já desvantagens temos a complexidade de programação, custo de hospedagem e desempenho.
