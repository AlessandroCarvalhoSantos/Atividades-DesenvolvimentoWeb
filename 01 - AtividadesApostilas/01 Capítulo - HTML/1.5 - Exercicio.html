Exercício:Qual é o caminho percorrido por uma requisição HTTP a uma página de hipertexto estática?

O caminho começa pela solicitação do usuário, na inserção da URL no navegador, o navegador interpreta a URL e determina o protocolo que vai ser usado geralmete HTTP ou HTTPS, o nome do servidor e o subcaminho dentro do servidor. Depois o navegador consulta a um servidor DNS qual o IP do servidor solicitado. Depois é estabelecido uma conexão com servidor e enviado uma requisição, o servidor então retorna uma resposta, é renderizado no navegador a resposta exibindo a resposta ao usuário final.
